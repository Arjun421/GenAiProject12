# Exam Question Analytics System - Milestone 1

## Overview
ML-based system that predicts exam question difficulty using classical machine learning and NLP techniques.

## Features
- Text preprocessing and TF-IDF feature extraction
- Logistic Regression and Decision Tree classifiers
- Difficulty categorization (Easy/Medium/Hard)
- Interactive Streamlit UI
- Model performance metrics

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

## Technical Stack
- **ML Models**: Logistic Regression, Decision Tree
- **NLP**: TF-IDF Vectorization
- **Framework**: Scikit-Learn
- **UI**: Streamlit

## Metrics
- Accuracy
- Precision
- Recall
- Classification Report
